# 🌌 Aura 

> **灵感不隐匿，交互有光芒。**

Aura 是一款专注于**个性照片添加**与**个性化内容展示**的 Android照片墙应用。在这里，我们通过现代化的设计语言，探索移动设备交互的新可能。

---

### ✨ 核心亮点

#### 🧊 视觉：磨砂玻璃美学
拒绝平庸的色块。Aura 深度定制了具有真实物理质感的**Google系风格（Frosted Glass）界面**，让每一次滑动都伴随着光影的律动。

#### 🛠️ 交互：系统实验室
打破原生的束缚。我们集成了一系列实用的系统辅助功能：
* **界面强制适配**：让那些不听话的应用也乖乖服从你的显示偏好。
* **自定义优化**：针对你的设备进行底层逻辑微调，运行更丝滑。
* **极简主义**：干净、纯粹，没有任何冗余的广告与视觉干扰。

#### 💎 技术基因
采用最新的 **Jetpack Compose** 引擎驱动，这意味着更流畅的动画响应和更稳定的性能表现。
